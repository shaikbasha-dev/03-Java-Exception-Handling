# 🚀 How to Use

1. Clone this repository.
2. Navigate to the **Source-Code** folder.
3. Open the desired Java program.
4. Compile the program.

```bash
javac FileName.java
```

5. Run the program.

```bash
java ClassName
```

6. Observe the output and study the comments to understand the concept.
